# Customer Churn Prediction Engine
A machine learning pipeline developed to forecast customer attrition using the Telco Churn dataset.

### Technologies Used
- Python, Scikit-Learn, Pandas, NumPy, SMOTE, GridSearchCV

### Key Features
- Implemented SMOTE to handle class imbalance (7,000+ records).
- Optimized model performance using GridSearchCV for hyperparameter tuning.
- Achieved 78.99% accuracy and 63.73% F1-Score.
